# 🚗 Car Dealership Review — IBM Full Stack Developer Capstone
This is a full-stack web application that allows users to browse car dealerships across the United States and read or submit reviews for each location. 

# Project Overview
This capstone project was built as the final requirement for the IBM Full Stack Software Developer Professional Certificate. It demonstrates end-to-end development skills by combining a React frontend, an Express/Node.js backend, a Django-based administration layer, and cloud deployment via IBM Kubernetes Service.
